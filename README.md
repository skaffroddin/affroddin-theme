# Affroddin WordPress Theme

## Overview
**Affroddin** is a custom WordPress theme developed by **Sekh Affroddin**, a passionate web developer. This theme is designed for a personal portfolio website, showcasing web development projects, skills, and videos related to development tutorials.

It is built to be **responsive, modern, and developer-friendly**, making it easy to customize and extend.

---

## Features
- Fully responsive layout for all devices  
- Customizable header, footer, and homepage sections  
- Portfolio section to showcase projects  
- Integrated video section to share developer tutorials and content  
- Clean and lightweight code following WordPress best practices  
- Compatible with latest WordPress versions
